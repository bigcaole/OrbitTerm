# OrbitTerm（轨连终端）

OrbitTerm 是一款面向专业运维与安全连接场景的桌面终端工具，提供 SSH 多标签会话、可视化 SFTP、身份复用、端到端加密金库、AI 命令辅助与开发者诊断能力。

## 产品定位
- 安全：本地金库采用端到端加密，主密码仅在本机参与解密。
- 直观：主机、身份、终端、SFTP 与设置中心集中在统一界面。
- 智能：内置 AI 命令助手与 AI 故障解释，提升排障效率。

## 核心能力
- 金库与身份
- 主密码解锁、自动锁屏
- 主机与身份解耦（一次修改身份，可同步影响关联主机）
- 多步骤主机向导（基础信息、认证配置、高级选项）

- 终端与会话
- 多标签会话管理
- KeepAlive、跳板机链路支持
- 高吞吐输出限流，防止界面卡顿

- 文件与路径联动
- 可视化 SFTP（目录浏览、上传、下载、重命名、删除）
- 路径面包屑导航
- 终端与 SFTP 路径同步
- 在终端打开目录、直接运行脚本

- 智能与诊断
- AI 自然语言转命令
- SSH 报错一键“问问 AI 怎么修”
- Orbit Inspector 实时连接日志（含握手阶段信息）
- 启动健康检查（网络、数据目录权限、系统 WebView 运行时）

- 备份与更新
- 导出加密备份文件
- 内置更新检查与安装入口

## 适用场景
- 日常服务器运维与巡检
- 多环境主机统一管理（开发、测试、预发、生产）
- 使用跳板机的受限网络接入
- 需要统一身份复用与审慎变更的团队协作

## 快速上手（用户视角）
1. 首次启动：完成欢迎引导，设置高强度主密码。
2. 解锁金库：输入主密码进入仪表盘。
3. 添加主机：通过三步向导填写主机、认证与高级选项。
4. 建立会话：在主机列表点击“新建 Tab”。
5. 文件管理：在下方面板使用 SFTP 浏览与传输文件。
6. 智能协助：按 Cmd/Ctrl+K 打开 AI 助手生成命令。
7. 诊断排障：点击“查看连接日志”进入 Orbit Inspector。
8. 安全备份：在 Inspector 内导出加密备份文件。

## 安全说明
- 不保存你的主密码，不提供主密码找回能力。
- 导出的备份为加密数据，不包含明文主机与凭据。
- AI 生成命令可能存在风险，执行前请人工复核。

## 私有云同步（账号版）
- 支持邮箱注册、登录与 JWT 鉴权。
- 解锁金库后会弹出同步登录窗口（登录/注册/跳过），登录状态默认持久化。
- 客户端会在金库变更后自动上传加密 Blob，并在登录后自动拉取较新版本。
- 同步服务默认只接受 HTTPS 请求，避免明文传输凭据与加密包。
- 双端需使用同一主密码解锁金库；新设备首次登录建议先手动执行一次“立即拉取”再编辑。
